# Формат
Лабы распределены по папкам с темами. В каждой папке лежат (.pdf) со всеми условиями задачам по данной теме и типовые задачи в виде кода (.с) . 
В каждом файле с кодом в самом начале идёт общая теория по теме. Далее сам код с объяснением логики решения.

Нужно уточнить, что в теории не написаны асболютно все функции, мысли, идеи или другие виды бреда, которые могут быть использованы. Каждая теория раскрывает только новые необходимые методы, которых не было в прошлых темах. Рекомендуется изучать темы в последовательности от номера 1 до 6 по порядку.

13-ая лаба отличается тем, что в ней два вида решения. Первый вид - это обработка английских букв (папка Английский язык). Второй вид - это обработка русских букв (папка Русский язык). Русских букв с очень большой вероятностью не будет. Но всё же шанс есть. Поэтому разобраны сразу два вида для тех, кто хочет подстраховаться.
