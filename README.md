# Формат
Лабы распределены по папкам с темами. В каждой папке лежат (.pdf) со всеми условиями задачам по данной теме и типовые задачи в виде кода (.с) . 
В каждом файле с кодом в начале идёт общая теория по теме. Далее сам код с объяснением логики решения.
